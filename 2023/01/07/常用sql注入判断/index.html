常用sql注入判断

发表于 2023-01-07 分类于 SQL注入 阅读数： 本文字数： 604 阅读时长 ≈ 2 分钟

‘ OR 1=1

一般SQL代码中会有这么一段
SELECT * FROM TABLENAME WHERE ID = 变量，
当通过注入后会变成
SELECT * FROM TABLENAME WHERE ID = 变量 OR 1=1;
就会使WHERE后面的表达式变成一句可有可无的表达式，与
SELECT * FROM TABLENAME相等
然后就可以通过这种句式来取得当前数据表中所有的用户信息。
